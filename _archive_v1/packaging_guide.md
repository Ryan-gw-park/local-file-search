# 윈도우 사용자용 설치 및 빌드 가이드

이 가이드는 개발 지식이 전혀 없는 분들을 위해 작성되었습니다.
MS 오피스만 설치된 일반 PC에서도 따라하실 수 있습니다.

## 1단계: Python(파이썬) 설치하기
이 프로그램은 파이썬이라는 언어로 만들어졌습니다. 실행 파일을 만들기 위해 딱 한 번 설치가 필요합니다.

1.  [Python 공식 홈페이지(클릭)](https://www.python.org/downloads/)에 접속합니다.
2.  노란색 **"Download Python 3.x.x"** 버튼을 누릅니다.
3.  다운로드된 설치 파일을 실행합니다.
4.  ⚠️ **중요**: 설치 화면 맨 아래에 있는 **"Add Python.exe to PATH"** 체크박스를 **반드시 체크**해야 합니다. (이걸 안 하면 작동 안 함!)
5.  "Install Now"를 눌러 설치를 완료합니다.

## 2단계: 소스 코드 다운로드
1.  이 GitHub 페이지의 우측 상단에 있는 초록색 **<> Code** 버튼을 누릅니다.
2.  **Download ZIP**을 클릭하여 압축 파일을 받습니다.
3.  다운로드한 압축 파일을 바탕화면 등 편한 곳에 **압축을 풉니다**.

## 3단계: 원클릭 빌드 실행
이제 복잡한 명령어 입력 없이, 마우스 클릭만으로 실행 파일을 만들 수 있습니다.

1.  압축을 푼 폴더(`local-file-search-main`)로 들어갑니다.
2.  **`build.bat`** 파일을 찾아 더블 클릭합니다.
    *   (혹시 "Windows의 PC 보호" 창이 뜨면, "추가 정보" -> "실행"을 누르세요)
3.  검은색 창이 뜨고 자동으로 필요한 파일들을 설치합니다.
4.  화면에 "한국어 버전"을 만들지 물어보면 숫자 **1**을 입력하고 엔터를 칩니다.

## 4단계: 내 프로그램 확인하기
검은색 창에 "모든 작업이 완료되었습니다!"라고 뜨면 성공입니다.

1.  같은 폴더 안에 새로 생긴 **`dist`** 폴더로 들어갑니다.
2.  **`LocalFileSearchAI_KR.exe`** 파일이 보일 것입니다.
3.  이제 이 파일만 있으면, 인터넷이 없어도 어디서든 AI 검색기를 사용할 수 있습니다!
    *   바탕화면으로 복사해서 사용하세요.

## 5단계: (만약 에러가 난다면) 문제 해결
빌드 도중 빨간 글씨로 에러가 뜨거나 "Microsoft Visual C++ 14.0 is required"라는 메시지가 보이면 아래 내용을 따라하세요.
최신 버전의 Python을 사용할 때, 일부 프로그램(AI 관련)은 C++ 도구가 필요합니다.

1.  **[Microsoft C++ Build Tools 다운로드(클릭)](https://visualstudio.microsoft.com/visual-cpp-build-tools/)**
2.  "Build Tools 다운로드"를 눌러 설치 파일을 받고 실행합니다.
3.  설치 화면에서 **"C++를 사용한 데스크톱 개발"** (Desktop development with C++) 항목을 **체크**합니다. (왼쪽 위 체크박스)
4.  오른쪽 아래 **"Install"** (설치) 버튼을 누릅니다. (용량이 꽤 큽니다, 10분 이상 소요될 수 있음)
5.  설치가 다 끝나면 컴퓨터를 재부팅하고, 다시 `build.bat`을 실행하세요.
